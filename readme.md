# 1.1 What is CAM?

**Computer-Aided Manufacturing (CAM)** is the process of using software to convert a 3D model into machine instructions (G-code) that can be executed by CNC machines.  
CAM connects the digital world (CAD models) to the physical world (manufactured parts).

### CAM‚Äôs Responsibilities:
- Translating geometry into cutting moves  
- Selecting appropriate tools  
- Generating toolpaths  
- Simulating operations for safety  
- Exporting G-code via a post processor  

### Why CAM Exists:
Unlike manual machining‚Äîwhere a machinist manually turns dials or uses DRO‚ÄîCAM allows:
- Repeatability  
- Precision  
- Complex geometry creation  
- Automation of adaptive and intelligent toolpaths  
- Reduction of human error  

Without CAM, modern manufacturing would be impossible.

---

# 1.2 CAM Workflow Overview  

The CAM workflow follows this structure:

1. **Import or create the CAD model**  
2. **Define the stock** (raw material)  
3. **Choose the Work Coordinate System (WCS)**  
4. **Create toolpaths** using different operations  
5. **Simulate & verify** for gouges, collisions, air cuts  
6. **Post-process** into G-code  
7. **Transfer the code** to the CNC machine  
8. **Run the job** and inspect the finished part  

Every CAM system‚Äîfrom Fusion to Mastercam and NX‚Äîfollows these core steps.

### Fusion 360 CAM Strengths:
- Modern adaptive strategies  
- Built-in simulation  
- Cloud-connected tool libraries  
- Beginner-friendly workflow  
- Industry-grade post processors

### Key Principles:

#### **1. Every CAM project begins with a ‚ÄúSetup.‚Äù**
The Setup defines:
- Stock size  
- WCS  
- Orientation  
- Model selection  

Without a Setup, toolpaths cannot be created.

#### **2. Toolpaths belong to a Setup.**
Each operation references:
- The tool  
- Cutting parameters  
- Strategy (2D, 3D, drilling, etc.)  

#### **3. Order matters.**
Toolpaths execute top-to-bottom in the Browser.

Typical order:
- Adaptive clearing (roughing)  
- Pocketing  
- Contours  
- Finishing  

#### **4. Simulation is tied to each toolpath**
Simulation shows:
- Tool motion  
- Chip removal  
- Stock leftover  
- Collisions

# 1.5 Understanding the Manufacturing Timeline

Fusion 360 keeps a **history** of all operations.

- Roll back the history  
- Delete or edit operations  
- Update all toolpaths after a change  

### What updates the timeline?
- Changing stock  
- Editing geometry  
- Editing toolpaths  
- Modifying tools  

### Regeneration:
If Fusion detects changes, toolpaths show a ‚ö†Ô∏è icon and must be regenerated.

# 1.7 CAD vs CAM: What Matters

CAM depends heavily on **CAD quality**.

### Good CAD practices for CAM:
- Fully closed bodies  
- Proper fillets  
- Clean edges (no micro debris)  
- Avoid self-intersecting surfaces  
- Avoid duplicate bodies  
- Remove unnecessary internal features  

### What CAM needs from the CAD model:
- Accurate shape  
- Accurate dimensions  
- Clear manufacturing intent  
- Consistent face geometry  

### What CAM can ignore:
- Materials applied in CAD  
- Appearance and colors  
- Construction sketches

# 1.8 CAM Safety Basics

Before generating a single toolpath, machinists must follow professional safety principles.

### **1. Machine Environment**
- Keep spindle and tool area clear  
- Remove T-handles, vises keys  
- Verify no loose clothing or jewelry  

### **2. Toolpath Safety Checks**
Always simulate:
- Check for collisions  
- Watch clearances at retract heights  
- Look for sudden direction changes  

### **3. Proper Fixturing**
Part must not move during cutting:
- Torque vise correctly  
- Ensure parallels are clear  
- Confirm stock is seating flat  

### **4. Safe Speeds & Feeds**
Too aggressive ‚Üí tool breakage  
Too conservative ‚Üí rubbing and heat buildup  

### **5. Dry Run First**
Use:
- Air cutting  
- Slow feed override  
- Single block mode  

### **6. Emergency Stop Knowledge**
Know the difference between:
- Feed hold  
- Reset  
- E-stop
